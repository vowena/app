# Code of Conduct

## Our commitment

We want Vowena Dashboard to be a welcoming, respectful project for contributors of every background and experience level.

## Expected behavior

- Be kind, constructive, and specific.
- Assume good intent while still being honest about problems.
- Give feedback on code and ideas without attacking people.
- Share context when making decisions so others can learn.
- Respect maintainers, contributors, and users who are trusting us with financial software.

## Unacceptable behavior

- Harassment, threats, discrimination, or hate speech
- Personal attacks, dogpiling, or repeated bad-faith arguing
- Publishing private information without consent
- Sexualized language or imagery in project spaces
- Deliberately disruptive behavior that makes collaboration unsafe

## Scope

This applies in issues, pull requests, code review, discussions, chat, events, and any public space where someone is representing the project.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers privately through GitHub. If private GitHub contact is not possible, email `security@vowena.xyz` with the subject prefix `[conduct]` and we will route it appropriately.

## Enforcement

Maintainers may remove content, pause conversations, or restrict participation when needed to protect the community.
